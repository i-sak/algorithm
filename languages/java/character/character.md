# Java - Character

Java에서 Character 자료형에 대해 알아보자.

Character는 char 타입으로 사용되며 문자를 뜻한다.

char 타입에서 많이 찾아보는 아스키 코드

__아스키 코드 보기__
```java
public class Main {
    public static void main(String[] args) {
        // ascii code
        int i = (int) 'A';
        System.out.println(i); // 'A' => 65
    }
}
```
